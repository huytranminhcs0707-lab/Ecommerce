# 🛒 E-commerce Frontend (React Learning Project)

🚀 **Live Demo:** [Click here to view the live project](https://ecommerce-rust-delta.vercel.app/)

This project was developed primarily as a hands-on learning experience to master **React.js**. The main focus of this repository is the frontend architecture, state management, and API integration using React.

*Note: To concentrate fully on learning the React framework, the base HTML/CSS templates and the backend server (API) were provided by external sources. My core responsibility was to bring the static UI to life and handle the frontend logic.*

## 🎯 Project Objectives & Focus
- Understand and implement **Component-based architecture** in React.
- Manage local and global state (e.g., handling shopping cart data, user sessions).
- Handle side effects and fetch data from a RESTful API using `useEffect` and `fetch`/`axios`.
- Implement client-side routing using **React Router**.
- Convert standard HTML/CSS into reusable React components (JSX).

## ✨ Key Features (Frontend Implementation)
- **Product Display:** Fetch and render product listings dynamically from the backend API.
- **Navigation:** Smooth client-side routing between home, product details, and cart pages.
- **Interactive UI:** Handle user events, form submissions, and state changes seamlessly.
- **Cart Management:** Add, update, and remove items from the shopping cart based on component state.

## 🛠 Tech Stack

**Frontend Logic (My Contribution):**
- **React.js** (Core framework)
- **React Router** (Navigation)
- **Axios / Fetch API** (Data fetching)

**UI & Backend (External Sources):**
- **Styling:** Pre-made HTML5 & CSS3 templates.
- **Backend:** Provided RESTful API.
